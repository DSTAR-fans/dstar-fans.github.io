RS-BA1 Remote Control

About the RS-BA1 Software

The Icom RS-BA1 IP Remote Control Software allows users of select Icom radios to remotely control their radios from across the room, or across the world. While the upgrades are free downloads from the Icom website, the base software costs about $150 as of 2023-01-31.

At first I thought it seemed expensive, but it's one of the more affordable Icom accessories and the capabilities it unlocks are unsurpassed.

Using the RS-BA1 Software for D-STAR

While use of the RS-BA1 software for analog radio functions is well documented in the manual, I had some trouble finding instructions for how to use the D-STAR functions of my radios.

To access these instructions, open the RS-BA1 Remote Control app and select Help in the file menu (or press F1), then select the Search tab and search for DV.

If you just want to expierment without reading the manual, you can get to the DV menu by selecting View > DV in the file menu of the RS-BA1 Remote Cotnrol software. The Memory Channel Editor contains the list of repeaters, frequencies, and offsets; as well as Call Sign settings.

If you're familiar with manually setting YOUR, RPT1, and RPT2 values in your radio, this will be similar. If not, you may find the D-STAR Calculator hosted by dstarinfo.com to be helpful for determining these values.

These settings can be viewed from your radio in the CS or Call Sign menu at any time, but understanding how to set them manually will remove some of the mystery of D-STAR operations.

When editing the Call Sign (CS) values, it's a good idea to keep these D-STAR memonics in mind.
